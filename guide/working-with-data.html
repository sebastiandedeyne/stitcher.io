Stitcher Guide

Working with data

YAML data

Collection of data.

# src/data/collection.yml

entries:
    entry-a:
        title: Example Entry A
        intro: Lorem ipsum dolor sit amet
        body: entry-a.md
        image:
            src: img/blue.jpg
            alt: A Blue image
    entry-b:
        title: Example Entry B
        intro: This is the second entry
        body: entry-a.md
        image: img/orange.jpg
# site.yml

variables:
    enries: collection.yml

A file for one entry.

# src/data/entry-a.yml

entry-a:
    title: Example Entry A
    intro: Lorem ipsum dolor sit amet
    body: entry-a.md
    image:
        src: img/blue.jpg
        alt: A Blue image
# site.yml

variables:
    entryA: entry-a.yml

Markdown

// src/data/guide/index.md

## Title

Lorem ipsum dolor sit amet.
# site.yml

variables:
    body: guide/index.md

JSON

// src/data/collection.json

{
    "entries": {
        "entry-a": {
            "title": "A title",
            "body": "guide/helper-functions.md"
        }
    }
}
# site.yml

variables:
    entries: /collection.json
// src/data/entries/entry-a.json

{
    "entry-a": {
        "title": "A title",
        "body": "guide/helper-functions.md"
    }
}
# site.yml

variables:
    entryA: /entries/entry-a.json
